Телеграм-бот, который отвечает на вопросы основываясь на ваших документах.
Текст каждого документа пропускаете через модель эмбеддингов.
Полученные векторы сохраняете в ChromaDB вместе с исходным текстом и метаданными.

### Технические требования к функционалу. 

### 1. Строгое соответствие источникам.
Бот должен отвечать только на основе информации со слайдов.
Если ответа нет в слайдах, бот должен сообщить об этом.

### 2. Кэширование ответов. 
Бот не должен обращаться к LLM, если пользователь задаёт точно такой же вопрос, на который уже был дан ответ. 
Он должен использовать кэшированные ответы из предыдущих сообщений. Это снижает затраты и ускоряет ответ.
